# Hi, I'm Jay 👋

**Full-stack engineer building from the Flutter UI down to the Linux box it runs on.**

My niche is **real-time communications** — VoIP, WebRTC, and SIP telephony. I build production phone systems on FreeSWITCH, cross-platform softphones in Flutter, and the PHP backends and self-managed Debian infrastructure that keep them running. Currently leveling up into **cloud & platform engineering** (AWS, containers, IaC).

📍 Antipolo City, Philippines · 🛠️ Software Engineer @ RingQ (Cal4Care)

### 🔧 What I work on

- **Real-time comms** — multi-tenant PBX systems, SIP/WebRTC media, FreeSWITCH telephony
- **Cross-platform apps** — Flutter on mobile, Windows desktop, and web (incl. softphone with `sip_ua` + `flutter_webrtc`)
- **Backend & data** — Node, PHP (Laravel, Phalcon, CodeIgniter), PostgreSQL, Redis
- **Infra & ops** — Debian servers, Nginx, deployment automation, Fail2ban hardening, bash tooling
- **Now learning** — AWS, Docker/Kubernetes
